## ARM MOUNTED ON RC CAR

THIS PROJECT IS A GROUP PROJECT AND MY ROLE WAS TO MAKE ITS STRUCTURE AND DESIGN RC CAR CHASIS
THE COMPONENTS USED WERE:
- ARDUINO MEGA R3
- ESP 32 
- BO GEAR MOTORS
- SERVO MOTORS

THE TECHNOLOGIES AND SOFTWARE USED FOR THIS PROJECTS WERE:
- LASER PRINTING FOR CAR CHASIS
- 3D PRINTING FOR ARM AND VARIOUS PARTS
- FUSION FOR DIGITAL ASSEMBLY AND 3D MODEL CREATION
- AUTOCAD FOR CHASIS BASE DESGIN (REFRENCE OR PATH DESIGN FOR LASER PRINTING)
- ARDUINO IDE FOR PROGRAMING ESP AND MEGA R3
